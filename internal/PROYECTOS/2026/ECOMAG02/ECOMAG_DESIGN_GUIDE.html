4. Espaciado y Layout

Contenedor principal

  • Max width: 1280px (max-w-7xl)
  • Padding horizontal: 16px (movil) / 24px (tablet) / 32px (desktop)
  • Padding vertical secciones: 64px (movil) / 96px (desktop)

Grid system

  • Servicios: 1 col (movil) → 2 col (tablet) → 3 col (desktop)
  • Proyectos: Stack vertical (movil) → 2 col masonry (desktop)
  • Mision/Vision/Valores: 1 col (movil) → 3 col (desktop)
  • Contacto: 1 col (movil) → 5fr/7fr grid (desktop)
  • Footer: 1 col (movil) → 2 col (tablet) → 4 col (desktop)
